# README
# Tic-Tac-Toe

Python으로 구현한 콘솔 기반 틱택토(Tic-Tac-Toe) 게임입니다.
사용자는 `X`, 컴퓨터는 `O`를 담당하며, 컴퓨터는 빈 칸 중 하나를 무작위로 선택하여 둡니다.

---

## 게임 소개

- 3x3 보드에서 진행되는 전통적인 틱택토 게임
- 사용자가 먼저 `X`로 시작하고, 이어서 컴퓨터가 `O`로 무작위 위치에 둠
- 가로, 세로, 대각선 중 하나라도 같은 기호로 채워지면 승리
- 보드가 모두 채워졌는데 승자가 없으면 무승부

---

## 실행 방법

```bash
python tic_tac_toe.py
```

Python 3 버전이 필요합니다. (`random` 모듈은 표준 라이브러리이므로 별도 설치 불필요)

---

## 사용 방법

1. 프로그램을 실행하면 3x3 보드가 출력됩니다.
2. `행 번호`와 `열 번호`를 각각 **0, 1, 2** 중 하나로 입력합니다.
   - 예: 보드 정중앙에 두고 싶다면 행 `1`, 열 `1` 입력
3. 입력한 위치가 비어 있으면 `X`가 놓이고, 이어서 컴퓨터(`O`)가 자동으로 빈 칸 중 하나에 무작위로 둡니다.
4. 한 쪽이 승리하거나 무승부가 되면 게임이 자동으로 종료됩니다.

### 보드 좌표 예시
```
(0,0) | (0,1) | (0,2)
------+-------+------
(1,0) | (1,1) | (1,2)
------+-------+------
(2,0) | (2,1) | (2,2)
```

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| `print_board()` | 현재 보드 상태를 화면에 출력 |
| `check_winner(symbol)` | 가로/세로/대각선을 확인하여 해당 기호의 승리 여부 판단 |
| `is_draw()` | 보드가 모두 채워졌는지 확인하여 무승부 여부 판단 |
| `computer_move()` | 컴퓨터(`O`)가 빈 칸 중 하나를 무작위로 선택하여 착수 |
| 잘못된 입력 처리 | 범위를 벗어난 좌표, 이미 채워진 칸, 숫자가 아닌 입력에 대한 예외 처리 |

---

## 예외 처리

- 0~2 범위를 벗어난 좌표를 입력하면 `"그런 위치는 없습니다."` 메시지를 출력하고 다시 입력받음
- 이미 채워진 칸을 선택하면 `"이미 채워진 칸입니다."` 메시지를 출력하고 다시 입력받음
- 숫자가 아닌 값을 입력하면 `"숫자만 입력해주세요."` 메시지를 출력하고 다시 입력받음
